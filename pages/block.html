In HTML programming, a block-level element is any element that starts a new line and uses the full width of the page or container. A block-level element can take up one line or multiple lines and has a line break before and after the element.

Block elements appear on the screen as if they have a line break before and after them. For example, the
< p>, < h1>, < ul>, < ol>, < dl>, < pre>, < hr />, < blockquote>, and < address> elements are all block level elements. They all start on their own new line, and anything that follows them appears on its own new line.
